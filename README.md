# jpyc.fun

## How to contribute

reactです。詳細はネットで調べてください。

```bash
git clone git@github.com:UnofficialClubJPYC/jpyc.fun.git
cp .env.sample .env    # .envでbuild directoryを変更しています
npm i # パッケージインストール
# ごにょごにょ
npm run start # localテスト環境起動
npm run build # upload用のファイルをbuild
```
